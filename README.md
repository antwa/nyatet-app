<h1 align="center">Todo List & Note Application</h1>

## Tech

-   Laravel 10
-   Laravel Turbo
-   Alpine JS
-   Tailwind CSS
-   Daisy UI
-   Feather Icons

## Fitur

-   Login & Register
-   Reset Password (dikirim ke email)
-   Todo List
-   Note (bisa dikunci)
-   Enkripsi Laravel menggunakan OpenSSL untuk menyediakan enkripsi AES-256 dan AES-128
